3. Data we process

We only process data that is necessary to provide and improve Quizly.

3.1 Account data

  • Username
  • E-mail address
  • Password (stored only as a salted and hashed value)

3.2 Usage and quiz data

  • Created quizzes (title, description, questions, answers)
  • Associated YouTube URLs used to create quizzes
  • Time of creation and updates of your quizzes

3.3 Technical data

When using Quizly, technical data may be processed automatically in order to provide the service securely and reliably. This can include:

  • IP address (short-term for technical purposes)
  • Browser type and version
  • Date and time of access
  • Operating system and device information

4. Authentication and cookies

For authentication we use JSON Web Tokens (JWT), which are stored as HTTP-only cookies. These cookies cannot be accessed via JavaScript and are used solely to keep you logged in.

We use in particular:

  • access_token – short-lived token for authentication
  • refresh_token – longer-lived token to obtain a new access token

After logging out, these tokens are invalidated on the server side so that they can no longer be used.

5. YouTube and external services

5.1 YouTube

Quizly allows you to create quizzes based on YouTube videos by entering a YouTube URL. The video itself is not stored by us. We download only the audio track of the video temporarily in order to create a transcript and generate quiz questions.

Please note that YouTube is provided by Google Ireland Limited. The processing of data by YouTube is governed by YouTube’s own privacy policy.

5.2 Audio processing and transcription

To extract and process the audio of a YouTube video, we use yt-dlp and FFmpeg.

For transcription of audio we use Whisper AI. In our setup Whisper is executed locally on the server. The audio and resulting transcript are used only to generate the quiz and are not shared with other users.

5.3 Quiz generation with Gemini

Quiz questions are generated using Gemini Flash (Google). For this purpose, the text transcript is sent to the Gemini API. The quiz (title, description, questions and answer options) is then stored in our database under your account.

The processing of data by Google and Gemini is subject to Google's own privacy terms and API terms of use.

6. Purpose of data processing

We process your data for the following purposes:

  • Providing the Quizly platform and its core features
  • Creating and managing your user account
  • Generating quizzes based on YouTube content you provide
  • Saving and displaying your created quizzes
  • Ensuring the security and stability of the application

7. Data retention

We store your personal data only for as long as necessary to fulfil the purposes described above or as required by law.

  • Account data is kept for the duration of your account’s existence.
  • Quizzes remain stored until you actively delete them or delete your account (if such a function is provided).
  • Technical log data may be stored for a short period for security and error analysis.

8. Sharing of data

We generally do not share your personal data with third parties unless:

  • you have given explicit consent,
  • it is required by law or authorities, or
  • it is technically necessary to use external services such as Gemini or YouTube (as described above).
